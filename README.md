## Comparison Table

| Dimension      | Vibe Version (v0)          | Pair Version (Manual / Copilot) | Verdict     |
| -------------- | -------------------------- | ------------------------------- | ----------- |
| Speed          | Generated in ~5–10 minutes | Took ~30–45 minutes             | Vibe faster |
| Control        | No control over structure  | Full control over logic         | Pair better |
| Code Quality   | Large auto-generated code  | Clean modular components        | Pair better |
| Explainability | Hard to understand         | Easy to explain                 | Pair better |
| Editability    | Hard to modify             | Easy to extend                  | Pair better |

## My Observations

* Time taken (Vibe): 10 minutes
* Time taken (Pair): 40 minutes

### What I liked about Vibe:

* Very fast to generate app
* UI is ready instantly

### Problems in Vibe:

* Code is difficult to understand
* Hard to modify or debug

### What I liked about Pair Programming:

* Full control over code
* Easy to understand logic
* Better structure using components

### Challenges:

* Took more time
* Required logical thinking
